Known Gotchas

1. Callback URL: use http://127.0.0.1 not http://localhost (X's OAuth2 validation treats them differently) 2. CRC challenge: must return {"response_token": "sha256=..."} Any other response = endpoint unverified = no events delivered 3. E2EE messages: DON'T call /2/dm_events/{id} for XChat messages Content is ONLY in the Activity Stream event payload (data.payload.encoded_event) 4. Legacy DM endpoint stops after E2EE upgrade Migrate to: Activity API + chat.received subscription 5. state.json contains private keys — NEVER commit to git Add to .gitignore immediately after xchat-bot-python login
